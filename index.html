add inglish to your website

add a floating translate button to any webpage with a single script tag. when visitors click it, all text on the page is translated to inglish. clicking again restores the original english.

quick start

paste this line before the closing </body> tag of your HTML:

<script src="https://inglish.us/translate-button.js"></script>

what it does

  • adds a small "inglish" button in the bottom-right corner of the page
  • loads the dictionary on first click (~2.7 MB, cached by the browser after that)
  • translates all visible text to inglish spelling, preserving capitalization
  • click again to switch back to english
  • skips text inside <code>, <pre>, <script>, <style>, <textarea>, and <input> elements

how it works

the script fetches the inglish dictionary (126,000+ words) from inglish.us and does word-by-word lookup on every text node in the page. HTML tags, attributes, and page structure are left untouched. only the visible text changes.

self-hosting

if you prefer to host the files yourself, download translate-button.js and dikshuneree.json, then edit the DICT_URL variable at the top of the script to point to your copy of the dictionary.

example

here is a minimal HTML page with the translate button:

<!DOCTYPE html>
<html>
<head>
  <title>my page</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>The quick brown fox jumps over the lazy dog.</p>
  <script src="https://inglish.us/translate-button.js"></script>
</body>
</html>
