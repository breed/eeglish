# ingglish alphabet

the ingglish alphabet maps every sound in american english to exactly one spelling. no silent letters, no ambiguity.

## rules

1. all letters and letter combinations have exactly one sound
2. a vowel by itself makes its common short sound
3. a vowel followed by `e` sounds like its name (long vowel)
4. a doubled vowel makes an elongated sound

## vowels

| IPA | ingglish | rule | example (english → ingglish) |
|-----|---------|------|-----------------------------|
| æ   | a       | short a (vowel alone) | cat → kat |
| ɛ   | e       | short e (vowel alone) | bet → bet |
| ɪ   | i       | short i (vowel alone) | bit → bit |
| ɔː  | o       | short o (vowel alone) | dog → dog |
| ʌ   | u       | short u (vowel alone) | but → but |
| ə   | u       | schwa (≈ short u)     | about → ubout |
| eɪ  | ae      | a + e = name of a | make → maek |
| iː  | ee      | e + e = name of e | see → see |
| aɪ  | ie      | i + e = name of i | bike → biek |
| oʊ  | oe      | o + e = name of o | go → goe |
| juː | ue      | u + e = name of u | use → uez |
| ɑ   | aa      | doubled a = elongated | hot → haat |
| uː  | oo      | doubled o = elongated | food → food |
| ʊ   | uu      | doubled u = elongated | book → buuk |
| aʊ  | ou      | diphthong | out → out |
| ɔɪ  | oi      | diphthong | boy → boi |
| ɝ   | yr      | r-colored vowel | bird → byrd |
